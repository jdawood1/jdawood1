## Featured Project

<table>
<tr>
<td>

### LLM Chatbot Platform — Senior Capstone

Production-style LLM chatbot platform with a **TypeScript/Node.js backend**, **AWS deployment workflow**, and **WordPress integration**.

**Tech:** TypeScript, Node.js, Express, Amazon Bedrock, Knowledge Bases/RAG, Guardrails, Docker, GitHub Actions, AWS ECS/Fargate, ECR, S3, CloudFront, React, Vite, Tailwind CSS, WordPress

**Highlights**

- Built a backend API for invoking an Amazon Bedrock Agent backed by Knowledge Base/RAG and Guardrails.
- Designed secure public-web embedding patterns using CORS/origin allowlisting and request validation.
- Automated CI/CD with GitHub Actions for linting, testing, Docker builds, ECR pushes, and ECS/Fargate staging deployments.
- Containerized the API for production-like smoke testing before releases.
- Built an embeddable React/Vite/Tailwind CSS chatbot UI delivered through a sandboxed WordPress plugin.
